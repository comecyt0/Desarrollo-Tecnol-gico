# Imágenes del carrusel del login

Coloca aquí las imágenes de los slides que se muestran en `/login`.

**Formato recomendado:**
- JPG o WebP (mejor compresión que PNG para fotografías)
- 1200×800 px (ratio 3:2)
- < 200 KB cada una

**Cómo referenciarlas:**
1. Sube el archivo a esta carpeta (ej. `slide-1.jpg`)
2. Entra al sistema como admin → `/admin/carrusel`
3. Edita un slide y en "URL de imagen" pon `/carrusel/slide-1.jpg` (ruta relativa)

El sistema sirve estos archivos desde el mismo dominio — **no se hace ningún
request a CDNs ni servicios externos** (Unsplash, Picsum, etc.).

Si no subes imágenes, los slides usan únicamente el gradiente institucional
(vino + dorado) con un mesh decorativo — visualmente correcto y 100% local.
